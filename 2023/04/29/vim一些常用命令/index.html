vim一些常用命令

vim一些常用命令：标注常用的为经常用到的

0、插入：i 按esc退出编辑，然后按”shift+：”可以输入一些命令，如不保存强制退出输入”q!”，保存退出输入”wq” （常用）

退出编辑 : esc

1、水平打开新文件：:sp 路径

2、垂直打开新文件：:vs 路径

3、改变窗口大小：:vertical res 80

4、跳到光标所在行末尾进行插入：A

5、跳到光标所在行开头进行插入：I

6、在光标所在行的上一行进行插入：0

7、在光标所在行的下一行进行插入：o

8、剪切整行：dd （常用）

9、多行剪切：ndd，其中n代表从光标所在行的往下数到第n行剪切 （常用）

10、复制整行：yy （常用）

11、粘贴：p （常用）

12、多行复制：nyy，其中n代表从光标所在行往下数到第n行复制

13、撤销上一步操作：u （常用）

14、恢复上一步被撤销的操作：ctrl r

15、在vim中有3中方法可以跳转到指定行（首先按esc进入命令行模式）：

1）ngg/nG （跳转到文件第n行，无需回车）

2）:n （跳转到文件第n行，需要回车）（常用）

3）vim +n filename （在打开文件后，跳转到文件的第n行）

16、到第一行或最后一行

1）底线命令模式：（常用）

:0或:1跳到文件第一行 （需要回车）

:$跳到文件最后一行 （需要回车）

2）命令模式

gg跳到第一行

shift+g跳到文件最后一行

17、块选择 （常用）

在命令模式下，块选择的按键意义：

v 字符选择，将光标经过的字符选择

V 行选择，将光标经过的行选择

[Ctrl]+v 矩形选择，可以用矩形的方式选择数据

y 将选中地方复制起来

d 将选中地方删除

v V [Ctrl]+v 按下后通过 上 下 左 右 选择按键进行选择

18、查找字符串（常用）

1）向光标之下：按”/“键

在命令模式下 输入/ ，然后输入你需要查找的字符串即可。向光标之下寻找你所输入的字符串。

可以使用n继续向下查找下一个字符串。

可以使用N向上查找字符串。

2）向光标之上：按”?”键

在命令模式下 输入? 然后输入你需要查找的字符串即可。向光标之上寻找你所输入的字符串。

注意：与/不同，

可以使用n继续向上查找下一个字符串。

可以使用N向下查找字符串。

每次查找完成后高亮并不会消失，可以在命令行模式输入：nohlsearch 或者：noh ，

也可以输入：set nohlsearch，设置不用高亮查询会让你的查询不会高亮显示，

如果需要高亮显示可以输入：set hlsearch。

19、分屏

ctrl+w : 返回原来窗口

退出：在命令模式下按照正常方式退出

2） 分屏打开任意文件

#命令（上下分屏幕）：

:split [FILENAME]

#或

:sp [FILENAME]

#命令（水平分屏幕）：

:vsplit [FILENAME]

#或

:vs [FILENAME]

20、全选（高亮显示）：按esc后，然后ggvG或者ggVG

全部复制 ：按esc后，然后ggyG

全部删除 ：按esc后，然后dG

解析：

gg ：是让光标移到首行，在vim才有效，vi中无效。

v ：是进入Visual（可视）模式。

G ：光标移到最后一行。

选中内容以后就可以其他的操作了，比如：

d ：删除选中内容。

y ：复制选中内容到0号寄存器。

“+y ：复制选中内容到＋寄存器，也就是系统的剪贴板，供其他程序用。

21.Vim中使用鼠标功能

1）打开鼠标功能（命令模式下）

:set mouse=a

2）关闭鼠标功能（命令模式下）

:set mouse-=a

3）鼠标模式

‘mouse’的参数说明：

// n 普通模式

// v 可视模式

// i 插入模式

// c 命令行模式

// h 在帮助文件里，以上所有模式

// a 以上所有模式

// r 跳过|lit-enter|提示

// A 在可是模式下自动选择

22.到行首，到行尾

vi到行首, 在esc模式下

按 “0″ , 是数字0, 就可以到达行首

到行尾是, 在esc模式下

按 “$“ , 就是shift+4

23.查找/替换字符串

按esc进入命令模式，按“：”

:s 命令来替换字符串。

：s/vivian/sky/ 替换当前行第一个 vivian 为 sky （常用）

：s/vivian/sky/g 替换当前行所有 vivian 为 sky （常用）

：n,$s/vivian/sky/ 替换第 n 行开始到最后一行中每一行的第一个 vivian 为 sky （常用）

：n,$s/vivian/sky/g 替换第 n 行开始到最后一行中每一行所有 vivian 为 sky （*常用）

n 为数字，若 n 为 .，表示从当前行开始到最后一行

:x,ys/vivian/sky/ 替换第 x 行开始到第 y 行中每一行的第一个 vivian 为 sky （常用）

:x,ys/vivian/sky/g 替换第 x 行开始到第 y 行中每一行所有 vivian 为 sky （*常用）

:.,+2s/vivian/sky/ 替换当前行”.”与接下来两行”+2”行中每一行的第一个 vivian 为 sky

:.,+2s/vivian/sky/g 替换当前行”.”与接下来两行”+2”行中每一行所有 vivian 为 sky

：%s/vivian/sky/（等同于 ：g/vivian/s//sky/） 替换每一行的第一个 vivian 为 sky （常用）

：%s/vivian/sky/g（等同于 ：g/vivian/s//sky/g） 替换每一行中所有 vivian 为 sky （*常用）

可以使用 # 作为分隔符，此时中间出现的 / 不会作为分隔符

：s#vivian/#sky/# 替换当前行第一个 vivian/ 为 sky/

：%s+/oradata/apras/+/user01/apras1+ （使用+ 来 替换 / ）： /oradata/apras/替换成/user01/apras1/

24.复制 单个 单词：

命令模式下：

移动光标到单词词首，快速按 “y+w”

25.光标移动——在单词间移动

命令模式下：

1）移动到下一个单词头部：w

2）移动到前一个单词头部：b

3）移动到下一个单词尾部：e

4）移动到前一个单词尾部：ge

5）向后移动到以空格分隔的字符串头部：W

6）向前移动到以空格分隔的字符串头部：B

7）向后移动到以空格分隔的字符串尾部：E

8）向前移动到以空格分隔的字符串尾部：gE

26.vim代码提示补全

在插入模式下的ctrl-p和ctrl-n组合键，不仅可以在插入模式下触发VIM的自动补全，

而且还可以用它们在补全列表中反向或者正向选择。这两个组合键均会调用关键字自动补全，

还有其他几种自动补全功能。

CTRL+n 普通关键字

CTRL+n CTRL+x 当前缓冲区关键字

CTRL+x CTRL+i 文件关键字

CTRL+x CTRL+] 标签文件关键字

CTRL+x CTRL+k字典查找

CTRL+x CTRL+f 文件名补全

CTRL+x CTRL+o全能（Omni）补全

27.自动对齐：（gg=G）

在命令模式下（即非“插入”等编辑模式），先输入gg，这时候光标会移动到第一行第一个字符，然后按 “=” 号

之后切换成大写，再按一下G，这时候光标会移到最后一行的第一个字符，这时候就可以看到代码被排得整整齐齐了！

28.vi编辑器从一个文件复制到另一个文件

1)先打开这两个文件：vi a1.log a2.log

2)：set number 显示行号

3)：100,200y(复制100行~200行的内容)

4)：n 切换到第二个文件

5)p(粘贴即可)，将光标移到需要复制的地方，按下P按钮即可。或者：600（切换到第600行），再按下p进行粘贴。

：1跳到行首

：$跳到最后一行
